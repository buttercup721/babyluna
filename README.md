# VC_luna

루나 한 명만을 위한 GitHub Pages 기반 웹 육아 보드 프로젝트입니다.

이 저장소는 원래 안드로이드 네이티브 앱으로 시작했지만, 현재 사용 기준의 메인 타깃은 `web/` 폴더에 있는 정적 웹 앱입니다. 서버 없이 브라우저에서 바로 동작하고, 데이터는 기기 로컬 저장소(localStorage)에 보관됩니다.

## 현재 기준 버전

- 메인 타깃: GitHub Pages 배포용 웹 앱
- 대상: 루나 한 명만을 위한 맞춤형 육아 보드
- 사용 연령대: 약 15개월~24개월
- 저장 방식: 브라우저 localStorage
- 플랫폼: 모바일 브라우저 우선, 데스크톱 대응

## 웹 앱에 들어간 핵심 기능

- 루나 전용 첫 설정 화면
- `오늘 / 기록 / 도움` 3탭 구조
- 빠른 기록: 식사 / 낮잠 / 기저귀 / 외출
- 최근 기록 목록
- 기록 상세 보기, 수정, 삭제
- 루나 프로필 편집
- 도움 카드 목록 / 검색 / 카테고리 필터
- 도움 카드 상세 보기
- 오늘 화면에 도움 카드 고정
- 앱 재방문 시 기록과 프로필 유지

## GitHub Pages 배포

배포 대상은 `web/` 폴더입니다.

1. 이 저장소를 GitHub에 올립니다.
2. 기본 브랜치를 `main` 으로 사용합니다.
3. 저장소의 Pages 설정에서 소스를 `GitHub Actions` 로 선택합니다.
4. `.github/workflows/deploy-pages.yml` 이 자동으로 `web/` 폴더를 배포합니다.

워크플로 파일: [deploy-pages.yml](./.github/workflows/deploy-pages.yml)

## 로컬 확인

별도 빌드 도구 없이 정적 파일만으로 동작합니다.

- 진입 파일: `web/index.html`
- 문법 확인: `node --check web/app.js`
- 빠른 스모크 테스트: 브라우저 없이 초기 렌더 경로 실행 확인 완료

## 참고 사항

- 현재 웹 버전은 서버 없이 한 기기에서 사용하는 개인 맞춤형 앱을 전제로 합니다.
- 여러 보호자 간 실시간 동기화 기능은 아직 없습니다.
- 안드로이드 앱 코드는 남겨두었지만, 앞으로 GitHub Pages 기준 작업은 `web/` 을 중심으로 이어가면 됩니다.
